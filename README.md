# Projeto Spring com Dois Bancos de Dados

Este projeto é um exemplo simples de como configurar uma aplicação Spring para utilizar dois bancos de dados diferentes.

## Sobre o Projeto

Este projeto foi criado como um estudo para entender como trabalhar com múltiplos bancos de dados em uma única aplicação Spring. A aplicação é capaz de alternar entre dois bancos de dados diferentes com base no ID fornecido na requisição.

## Tecnologias Utilizadas

- Spring Boot
- JPA / Hibernate
- PostgreSQL
